# Proyecto VUE CDN a VUE CLI

Tercera entrega del curso

### Consigna

Recupera tu 2do desafío genérico realizado en la clase 4 y conviértelo a VUE CLI.

### Aspectos a incluir en el entregable

Es la clase 4 realizaste un desafío genérico generando múltiples instancias de un componente con Vue CDN. Incluyendo pasaje de parámetros por props que cambiaban características de cada uno de los componentes que replicaste.
Toma ese mismo desafío y conviértelo a Vue CLI. Respetando las mismas funcionalidades y parametrizaciones por props aplicadas oportunamente.

## Para ejecutar proyecto

```
npm install
```

```
npm run serve
```
